## エグゼクティブサマリー

### 2026年1月-2月前半の運用成果

2026年1月〜2月8日の約5週間で、**381投稿**を実施し、**516,725インプレッション**を獲得。フォロワーは3,744名から4,196名へと**+452名（+12.1%）**の成長を達成しました。特に2月前半の勝率4.9%は過去最高水準であり、テーマを「AI活用」から「Threads運用ノウハウ」にシフトしたことが奏功しています。

### 主要トピック

**成功パターンの進化が明確に判明**
- 2025年10月の「AI活用」テーマから、2026年は「Threads運用ノウハウ」（投稿時間、長文vs短文、フック改善）に完全シフト
- 新フックパターン「あのー」型が平均3,528impで最強フックに浮上
- コメント欄展開型（メイン短文 + コメント4-5構成）が勝ちパターンとして定着
- 2月投稿のLINE導線到達率が7-8%に大幅改善（1月は1-2%）

**課題も鮮明に**
- メガバズの不在（最大19,632imp。2025年10月は270,000imp）
- 5K+投稿の勝率は3.4%（13/381件）と低水準が継続
- 投稿時間テーマへの偏重（TOP20中8件）による飽和リスク
- 金曜日のパフォーマンスが最弱（平均848imp、5K+投稿ゼロ）

**再現可能な成功の型を発見**
- 本レポートでは、2月前半の勝率4.9%を安定的に維持・向上させるための具体的な戦略と、今後14日間の実行プランを提示します。水曜日集中投稿、「あのー」パターン活用、コメント欄LINE導線の最適化が鍵となります。

**重大な発見**：

### 「あのー」パターンの圧倒的優位性

**トップ2は「あのー」パターン + 画像投稿**：

**4位：14,861 imp（1/27）- トピック設定不要論**
```
「あのー、Threads投稿するときにトピックは設定しない方が良いですよ」
```

**5位：14,561 imp（2/5）- 長文投稿の優位性**
```
「あのー、Threadsは短文で投稿しない方がいいですよ。」
```

**共通パターン**：
1. **「あのー」で始まるソフトな語り口**（親しみやすさ）
2. **画像付き投稿**との組み合わせ（スクショ・図解）
3. **リプライ数が多い**（14-18件）-- エンゲージメント向上に寄与
4. **「〜しない方がいいですよ」の否定形**で注意を引く
5. コメント欄で段階的に根拠と解決策を展開

### 警告・損失回避型の安定性

警告型は143件と最多投稿数。安定的に5K+投稿を生み出すワークホース。

**成功例（1位: 19,632 imp）**：
```
「Threads午後3時とか4時に投稿してる人、マジで損してます。」
```

**成功例（2位: 18,805 imp）**：
```
「Threadsで伸びやすい投稿時間を分析してない人、マジで損してます。」
```

**問題点**：
- 143件中5K+は7件のみ（勝率4.9%）
- 平均impは1,756と「あのー」パターンの半分
- 同一フレーズの繰り返しによる飽和リスク

**改善策**：
- 警告型は40%を維持しつつ、バリエーションを増やす
- 「あのー」パターンを20%まで増やす（現状3.1%）
- 画像・カルーセル投稿との組み合わせを強化

### 7つの勝ちパターン

**1. 「あのー」パターン + 画像投稿**
- 平均3,528imp（全フックタイプ中最高）
- 画像付きでリプライ率が高い（14-18件）
- 親しみやすい語り口でコミュニティ形成に寄与

**2. 「マジで損してます」警告フック**
- TOP30の約50%がこの型
- 損失回避バイアスを活用し、クリック率が非常に高い
- 投稿時間・長文・予約機能テーマとの相性が特に良い

**3. メイン短文 + コメント欄展開型の構成**
- メイン投稿（50-200字）でフック → コメント欄で段階的に深掘り
- コメント欄1: 体験談 + データ（300-500字）
- コメント欄2: 3ステップの具体策 + CTA（300-500字）
- コメント欄3: LINE導線リンク
- コメント欄4: 別投稿への誘導

**4. 具体的なデータ・数値の提示**
- 「閲覧数3倍」「9割の人が間違い」「フォロワー3ヶ月で3,100名」
- 抽象的な主張ではなく、具体的な数値で権威性を担保

**5. Threads運用ノウハウテーマに特化**
- TOP30のうち27件がThreads運用ノウハウ
- 投稿時間（8件）、長文投稿（5件）、曜日別（3件）が三大テーマ
- ターゲット層に直接刺さるテーマ選定

**6. 朝6-9時または夜18-21時に投稿**
- TOP30のうち朝6-9時: 7件、夜18-21時: 11件
- 合計18件/30件 = 60%がこの2つの時間帯

**7. 水曜日に集中投稿**
- 水曜日の平均imp 2,053（全曜日中最高）
- 投稿数が少ない（51件）ため競争が少なく、伸びやすい

## ボトルネック & リスク分析

### ボトルネック1：メガバズの不在

**問題**：
- 2026年の最大impは19,632（2025年10月は270,000）
- 10倍以上の差があり、フォロワー急増のドライバーが欠如

**原因**：
- テーマが「Threads運用ノウハウ」に特化し、マス向けのバズ要素が薄い
- AI活用テーマ（ChatGPT、Copilot等）からの撤退
- Threads運用テーマはターゲット層に刺さるが、非ターゲット層への拡散力が弱い

**解決策**：
- AI活用テーマを月2-3回再投入（旬のAIニュースとの組み合わせ）
- 「AI × Threads運用」の掛け合わせテーマを開発
- 企業事例 + 個人実践のテンプレートを活用

### ボトルネック2：投稿時間テーマへの偏重

**問題**：
- TOP20のうち8件が「投稿時間」テーマ
- 同一テーマの繰り返しによる飽和リスク

**原因**：
- 投稿時間テーマが安定的に伸びるため、つい同じテーマを繰り返す
- 新テーマの開拓が不足

**解決策**：
- 投稿時間テーマは週1回に制限
- 新テーマの開拓（コメント欄構成法、カルーセル活用法、シャドウバン対策、アルゴリズム解説等）
- 同じテーマでも切り口を変える（「投稿時間」→「2段階ピーク理論」「曜日×時間帯のマトリクス」等）

### ボトルネック3：金曜日のパフォーマンス低迷

**問題**：
- 金曜日の平均imp 848は全曜日中最弱
- 5K+投稿もゼロ

**原因**：
- 週末前の心理的な離脱（情報収集モードからリラックスモードへ）
- 競合投稿の増加？

**解決策**：
- 金曜日の投稿を最小限に（週1投稿以下）
- 水曜日（平均2,053imp、最強）に投稿を集中
- 金曜日は「週末に読むべきコンテンツ」の予告に限定

### ボトルネック4：画像・カルーセル投稿の不足

**問題**：
- カルーセル2件、画像15件のみ（全体の4.5%）
- 平均impはテキストより34-87%高い

**原因**：
- テキスト投稿の量産体制に最適化されている
- 画像・カルーセル作成のコストが高い

**解決策**：
- カルーセルを10%に増やす（スクショ、図解、Before/After）
- 「あのー」パターン + 画像の組み合わせを週2回導入
- Canvaテンプレートを作成し、制作コストを削減

### ボトルネック5：フォロワー増加ペースの鈍化

**問題**：
- 10月: +1,520名/月 → 1月: +382名/月 → 2月前半: +68名/8日

- Instagram・YouTubeとの連動でマルチチャネル流入を増やす
