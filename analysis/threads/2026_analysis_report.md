## 3. 伸びた投稿の共通パターン

### 3-1. 冒頭1行（フック）のパターン

TOP30投稿のフックを分類すると、以下の勝ちパターンが明確に存在する。

#### A. 警告・損失回避型（最強パターン）
- 「〜してる人、マジで損してます。」 -- **TOP30の60%がこの型**
- 「〜9割の人が間違えてます。」
- 「〜完全に損してます。」
- **平均imp: 約8,000-19,000**
- 人間の「損失回避バイアス」を突くため、クリック率が非常に高い

#### B. ソフト助言型（「あのー」パターン）
- 「あのー、〜しない方がいいですよ」
- **平均imp: 約7,800-14,800**
- 画像付き投稿との相性が抜群
- 親しみやすい語り口でリプライ数が多い（14-18件）

#### C. 数値インパクト型
- 「〜で、閲覧数3倍変わります。」
- 「〜10倍変わるので」
- **平均imp: 約5,500-18,700**
- 具体的な数値で読者の興味を引く

#### D. 対比型
- 「←〜できない人　〜できる人→」
- カルーセルとの相性が良い

### 3-2. 本文の構成パターン

**勝ちパターン: メイン短文 + コメント欄展開型**

| 要素 | 役割 | 文字数目安 |
|------|------|-----------|
| メイン投稿 | フックで興味を引く。「解説すると」で次を読ませる | 50-200字 |
| コメント欄1 | 体験談 + データ提示 + 「じゃあ具体的にどうすればいいかっていうと▼」 | 400-500字 |
| コメント欄2 | 3ステップの具体策 + 注意点 + CTA（フォロー誘導） | 400-500字 |
| コメント欄3 | LINE導線リンク | 短文 |
| コメント欄4 | 別投稿への誘導「これも必見です▼」 | 短文 |

**共通構造**:
1. 冒頭で「問題提起」（損してます系）
2. 自分の体験談（Before→After）で共感獲得
3. データ・数値で権威性担保
4. 3ステップの解決策で実践可能性を示す
5. 注意点で信頼性UP
6. CTA（フォロー + LINE）

TOP投稿のコメント欄は以下の統一的な構成を持つ:

| depth | 役割 | 内容 | 平均views |

| depth 0 | コメント欄1 | 体験談 + データ + 問題提起の深掘り | 親投稿impの15-20% |
